# FayeExtensions

Faye Ruby extensions for Medeo's real-time client.

## Installation

Add this line to your application's Gemfile:

    gem 'faye-extensions'

And then execute:

    $ bundle

Or build it yourself:

    $ gem build faye-extensions.gemspec

And install it as:

    $ gem install faye-extensions-x.x.x.gem
